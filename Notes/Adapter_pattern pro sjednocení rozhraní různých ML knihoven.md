# 1. Popis
Adapter pattern slouží ke sjednocení nekompatibilních rozhraní různých ML knihoven, například umožňuje používat metody jako fit() a predict() přes různé backendy (scikit-learn, PyTorch) bez nutnosti upravovat klientský kód.

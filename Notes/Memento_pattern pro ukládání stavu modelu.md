# 1. Popis
Memento pattern umožňuje zachytit a uložit vnitřní stav ML modelu (např. váhy, stav optimizeru) bez porušení enkapsulace, což je užitečné pro tvorbu checkpointů a možnost obnovy modelu pro trénink nebo nasazení.
